# SketchRPG
Java实验——手绘RPG


### 编译环境：

**Eclipse kepler版本**

jDK ： 1.7 verson

**运行方法**

1.Eclipse
	
    先import工程，然后点运行即可
    
	（由于资源文件夹res放在bin目录下，曾经出现过res文件夹莫名消失的情况，若出现该情况，请将与bin文件夹同层次的res放置到bin目录下再运行）

2.控制台
    
    打开cmd，cd到bin目录，输入 java SketchRPG 即可运行
